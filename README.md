# In-hand-Manipulator
This project aims to build a novel In-hand-Manipulation mechanism for its larger counterpart (Manipulator). 

## Timeline
* [ ] Designing the Modular Finger mechanism
* [ ] Designing Opening and closing Mechanism for the fingers
* [ ] Testing the Design and re-iterating
* [ ] Designing Control System for the in-hand Manipulation
* [ ] Final Conclusion once project is complete

## Motivation
* To be able to make an optimal and novelistic design and control system for the in-hand manipulation. 
* To be able to publish a paper through it.
